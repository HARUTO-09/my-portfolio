# Sai Varun Mukunda Portfolio

This is the personal portfolio website of Sai Varun Mukunda, an AI and Full-Stack Developer.

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Shadcn/ui

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Build for production: `npm run build`
